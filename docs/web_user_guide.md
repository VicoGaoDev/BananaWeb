# Web 端功能使用说明

本文档面向普通用户，帮助你快速了解 Web 端可以做什么、怎么用。

如果你需要不同长度或不同展示方式的版本，也可以直接查看：

- `docs/web_user_guide_one_page.md`：更短的 1 页版用户手册
- `docs/web_user_guide_screenshot.md`：按步骤拆分的截图说明版操作文档

## 1. 适用对象

这篇文档适合：

- 第一次使用 Web 端的用户
- 想快速了解各页面作用的用户
- 想知道常用操作入口在哪里的用户

## 2. Web 端能做什么

普通用户在 Web 端主要可以完成下面几类操作：

- 登录或注册账号
- 浏览首页内容和推荐入口
- 使用模板快速开始创作
- 进行 AI 生图
- 在无限画布中组织创作任务和自由节点（内部测试用户可见）
- 进行图编辑、局部重绘、提示词反推
- 查看历史记录
- 下载结果图
- 提交使用反馈
- 查看个人资料和积分记录

## 3. 主要页面说明

### 3.1 首页

首页是进入系统后的默认页面，主要用于：

- 快速了解平台的核心能力
- 进入生图页面开始创作
- 进入历史记录查看过往结果

如果你还不熟悉平台，建议先从首页进入，再按页面提示操作。

### 3.2 登录 / 注册

未登录时，可以先进行登录或注册。

- 已有账号：直接登录
- 没有账号：先注册，再登录使用

登录后，系统会开放更多和个人相关的功能，例如历史记录、积分记录、我的反馈、个人主页等。

### 3.3 模板中心

模板中心适合不知道怎么下提示词、或者想快速出图的用户。

你可以在这里：

- 浏览现成模板
- 查看模板效果
- 一键套用模板参数
- 跳转到创作页面继续修改后生成

如果你想更快上手，建议优先使用模板。

### 3.4 AI 生图页面

这是最核心的创作页面，常用功能都集中在这里。

#### 文生图

输入提示词后，选择模型、比例、尺寸等参数，即可发起生成任务。

适合：

- 从文字描述直接生成图片
- 快速测试创意方向

#### 图编辑

上传参考图后，可以结合提示词继续生成新的结果。

适合：

- 基于已有图片继续扩展
- 调整画面风格和表现方式

#### 局部重绘

上传原图后，在指定区域涂抹，再输入想修改的内容，即可对局部区域进行重绘。

适合：

- 修改人物、背景、细节
- 替换画面中的局部元素

#### 提示词反推

上传一张图片，系统会帮助你反推出适合 AI 绘画使用的提示词。

适合：

- 想参考某张图的提示词写法
- 不知道如何组织描述语句

#### 结果区

生成任务发起后，右侧会展示最近的任务结果。你可以直接进行：

- 预览图片
- 下载原图
- 基于结果图继续编辑
- 重新生成
- 提交反馈

### 3.5 无限画布

无限画布适合把多个创意方向放在同一个空间里持续探索。当前该功能处于内部测试阶段，仅管理员、超级管理员和白名单用户登录后可见。

你可以在这里：

- 新建、切换和重命名画布项目
- 在画布中提交文生图或图编辑任务
- 拖动画布、缩放画布、移动和拉伸节点
- 从画布图片中选择参考图继续编辑
- 添加自由文本节点和上传图片节点
- 搜索节点并聚焦到对应位置
- 使用一键整理按文本、上传图片、任务卡片横向排布

画布中的生图任务和普通 AI 生图使用同一套积分、并发和任务处理链路。提交任务前会展示预计消耗积分；积分不足时会提示购买积分。

## 4. 历史记录

历史记录页面用于查看你过去的任务结果。

你可以在这里：

- 按提示词筛选记录
- 查看任务详情
- 查看完整参数
- 重新编辑并再次生成
- 下载图片
- 对结果提交反馈
- 批量下载部分结果

如果你经常复用以前的作品，历史记录会非常有用。

## 5. 我的反馈

如果你在使用过程中遇到问题，或者对生成结果有意见，可以通过反馈功能提交信息。

你可以：

- 查看自己提交过的反馈
- 查看反馈详情
- 了解反馈处理情况

这个功能适合问题追踪和效果反馈。

## 6. 个人主页

个人主页主要用于管理你的个人信息。

你可以在这里：

- 修改用户名
- 上传或更换头像
- 修改登录密码

如果你需要更新个人资料，直接进入个人主页即可。

## 7. 积分记录

积分记录页面用于查看积分变化情况。

你可以了解：

- 积分增加或减少的记录
- 每次变动的时间
- 对应的用途或来源

如果你想知道积分为什么减少，优先查看这个页面。

## 8. 推荐使用流程

如果你是第一次使用，建议按下面顺序操作：

1. 登录或注册账号
2. 先浏览模板中心，快速了解效果
3. 进入 AI 生图页面开始创作
4. 在结果区查看并筛选满意结果
5. 到历史记录查看完整参数和过往作品
6. 需要时下载图片、再次编辑或提交反馈

## 9. 使用小提示

- 不知道怎么写提示词时，可以先用模板，或者使用提示词反推
- 想微调图片局部内容时，优先使用局部重绘
- 想在已有图片基础上继续创作时，优先使用图编辑
- 重要图片建议及时下载保存
- 历史记录适合回看参数和复用旧作品
- 如果一个创作过程包含多轮参考、对比和分支，建议使用无限画布管理节点关系

## 10. 注意事项

- 部分功能需要先登录后才能使用
- 无限画布目前为内部测试功能，普通用户可能暂时看不到菜单入口
- 图片原图有保存时效，建议在有效期内及时查看和下载
- 若生成失败或效果不理想，可以重新生成或提交反馈

## 11. 常见问题

### 找不到以前生成的图片怎么办？

先到`历史记录`中查看；如果图片比较重要，建议在生成后及时下载保存。

### 不知道从哪里开始？

建议先看`模板中心`，选一个接近你需求的模板，再进入创作页面继续修改。

### 生成结果不满意怎么办？

你可以尝试：

- 修改提示词后重新生成
- 使用图编辑继续调整
- 使用局部重绘修改局部内容
- 提交反馈说明问题
